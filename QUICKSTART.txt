SMART CITY DASHBOARD - QUICK START GUIDE
==========================================

📌 FASTEST WAY TO RUN:
----------------------
1. Open "index.html" in your web browser
2. That's it! The dashboard will work with demo data

🔑 FOR REAL WEATHER DATA (Optional):
------------------------------------
1. Get a FREE API key from: https://openweathermap.org/api
2. Open "script.js" file
3. Find line 3: WEATHER_API_KEY: 'YOUR_API_KEY_HERE'
4. Replace 'YOUR_API_KEY_HERE' with your actual key
5. Save and refresh the browser

✨ FEATURES YOU CAN USE:
-----------------------
- Select different cities from the dropdown menu
- View real-time weather and air quality data
- See energy consumption charts
- Check traffic density by zone
- Monitor waste management statistics
- Watch temperature trends over 24 hours
- Auto-refresh every 5 minutes

🎯 ALL REQUIREMENTS MET:
------------------------
✓ Real-time/Public API data
✓ Air quality, traffic, weather, energy, waste data
✓ Charts using Chart.js
✓ Clear and simple UI
✓ Multiple APIs (Weather + Air Quality)
✓ Real-time data updates
